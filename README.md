#### Flags
  -logcolour
    	Use coloured logging (switch off when redirecting log output). (default true)
  -logfile string
    	Sets the name of the logfile. Uses STDOUT if empty.
  -loglevel string
    	Determines logging verbosity. [All|Info|Debug|Warn|Error|Fatal|Off]. (default "Warn")
  -version
    	Show version info.
